# 货币资金与交易性金融资产

## 货币资金

包括库存现金、银行存款、其他货币资金

### 库存现金
#### （一）先进的备用金制度
企业实现付给内部部门或业务人员一笔一定金额的现金，供其零星开支使用，称为备用金。

!!! Note
    报销金额小于备用金：

    借：管理费用（报销金额）    
    &nbsp;&nbsp;&nbsp;&nbsp;库存现金（差额）    
    &nbsp;&nbsp;&nbsp;&nbsp;贷：其他应收款（备用金）    

!!! Note
    报销金额大于备用金：

    借：管理费用（报销金额）    
    &nbsp;&nbsp;&nbsp;&nbsp;贷：其他应收款（备用金）    
    &nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;库存现金（差额）    

!!! Note
    定额备用金：

    借：管理费用（报销金额）    
    &nbsp;&nbsp;&nbsp;&nbsp;贷：库存现金（报销金额）

#### （二）现金的清查

1. 清查的目的：账实相符
2. 清查方法：盘点
3. 清查结果的会计处理
    1. 盘盈：①借：库存现金 贷：待处理账户 ②借：待处理 贷：营业外收入
    2. 盘亏：①借：待处理账户 借：库存现金 ②借：其他应收款/管理费用（无法查明原因） 贷：待处理

### 银行存款
银行存款的期末检查

1. 方法：每月月末，将企业“银行存款日记账”的余额，与银行送来的账单核对。
2. 余额不一致原因
    1. 企业与银行任何一方的记账错误
    2. **未达账项**：由于凭证接受时间不一致而导致有些账还没记。
3. 具体操作：编制银行存款余额调节表
4. 未达账项不能以银行存款来记入

### 交易性金融资产的期末计价————市价法

#### 含义
每期期末，将金融资产调整为按市场计价。

#### 方法
设置“公允价值变动损益”账户

期末应调整的金额 = 市价 - 成本

#### 会计处理
若市价-成本>0，则   
借：交易性金融资产————公允价值变动
贷：公允价值变动损益

若市价-成本<0，则   
借：公允价值变动损益
贷：交易性金融资产————公允价值变动
